# sql-ecommerce-practice
A beginner-friendly MySQL practice project based on an E-Commerce database. This repository includes SQL queries covering database creation, table creation, constraints, filtering, operators, NULL handling, pattern matching, and other fundamental SQL concepts. Created to strengthen practical SQL skills for data analysis and database management.

This project contains MySQL practice queries using an E-Commerce database.

## Topics Covered

* Database Creation
* Table Creation
* Constraints
* Primary Key
* Foreign Key
* INSERT Statements
* DISTINCT
* WHERE Clause
* Comparison Operators
* Arithmetic Operators
* Logical Operators
* IS NULL / IS NOT NULL
* IN / NOT IN
* BETWEEN / NOT BETWEEN
* LIKE / NOT LIKE
